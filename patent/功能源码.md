# **二、设计 Token 美化渲染核心源码**

## **ai/flows/generate_project/steps/applyProjectDesignTokens.ts（节选）**

```TypeScript
/** 将 design-system.md 与当前 globals.css 合并为可运行的主题变量（Tailwind v4） */
function looksLikeGlobalsCss(text: string): boolean {
  const t = text.trim();
  return (
    /@import|@theme|@layer|@tailwindcss|@keyframes/.test(t) ||
    (t.includes("{") && /--color-|--font-/.test(t))
  );
}

// LLM 输出解析 → 写入 sites/{projectId}/app/globals.css
// 作为全站「美化渲染」单真源，下游 Page Agent 仅引用 CSS 变量与约定 class
```
