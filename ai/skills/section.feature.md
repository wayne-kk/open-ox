# Skill: Feature Section

Generate a feature/benefits section for landing pages.

## Structure

- **Section title**: Optional heading (h2)
- **Features**: 2–4 feature cards in a grid
- Each card: icon/title, description

## Layouts

- **Grid 2**: Two columns on desktop
- **Grid 3**: Three columns
- **Grid 4**: Four columns (icons only or icon+text)
- **Alternating**: Image left, text right, then swap

## Design Rules

- Consistent card height or flexible
- Icon: 12–16 size, muted or accent color
- Title: font-semibold
- Description: text-muted, text-sm or base
- Gap between cards: gap-6 to gap-8

## Placeholders

- `{{title}}` - Section title
- `{{features}}` - Array of { icon, title, description }
