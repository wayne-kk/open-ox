# Skill: Hero Section

Generate a hero section for landing pages.

## Structure

- **Headline**: Main value proposition (h1)
- **Subheadline**: Supporting copy (p)
- **CTA**: Primary and optional secondary buttons
- **Visual**: Optional image, illustration, or gradient background

## Variants

- **Centered**: Text and CTA centered, full-width
- **Split**: Text left, visual right (or vice versa)
- **Minimal**: Headline + CTA only, no subheadline

## Design Rules

- Headline: 2xl–4xl, font-bold
- Subheadline: lg–xl, text-muted
- CTA: Primary button prominent, secondary as outline/ghost
- Adequate padding (py-16 to py-24)
- Mobile: Stack vertically, reduce font sizes

## Placeholders

- `{{headline}}` - Main headline text
- `{{subheadline}}` - Supporting text
- `{{cta_primary}}` - Primary button label
- `{{cta_secondary}}` - Secondary button label (optional)
