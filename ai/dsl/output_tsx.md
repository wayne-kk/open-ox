# DSL: Output TSX

When the task requires React/Next.js component code, output TSX only.

## Rules

- No markdown code blocks (no ```tsx or ```)
- No explanation before or after
- Complete, runnable TSX
- Include all necessary imports
- Export the component (default or named)

## Format

- Use TypeScript
- Use Tailwind for styling
- Follow project conventions (see design_rules.md)

## Use Cases

- Section generation (hero, feature, cta)
- Layout generation (landing, blog)
- Component generation
