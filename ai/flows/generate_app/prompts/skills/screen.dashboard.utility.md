# Screen Skill: Utility Dashboard

Use this skill for productivity or utility apps where users need immediate control and status.

## Screen Intent

- Show what users can do now.
- Show current status/progress at a glance.
- Keep primary action always reachable.

## Recommended Regions

1. Compact status header (date/context/current state).
2. Main workspace panel (task list, records, controls).
3. Quick actions row (add, sync, start, edit).
4. Recent feedback/history strip.

## Design Rules

- Prioritize clarity over decoration.
- Use concise cards or list rows with clear affordances.
- Keep action semantics explicit (verb-first labels).
- Avoid hero-style promotional headings.
