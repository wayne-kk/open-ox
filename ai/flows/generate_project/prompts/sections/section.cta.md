## Section Prompt: Generate CTA Section

Generate a high-conviction call-to-action section that creates urgency and
focuses the user on one primary next step.

## Required Structure

1. A visually distinct background or panel.
2. A bold, benefit-focused headline.
3. Supporting copy with urgency or social proof.
4. Primary CTA and optional secondary CTA or email capture.
5. A micro-trust element such as a reassurance line or simple badge.

## Rules

- The CTA copy must match the page theme.
- Background contrast should clearly separate this section from adjacent sections.
- Use input state only if an email capture pattern is explicitly appropriate.
