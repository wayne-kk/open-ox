## Section Prompt: Generate Stats Section

Generate a stats or metrics section with impactful large numbers and compact explanatory labels.

## Required Structure

1. Optional short heading.
2. A grid of 3-5 metrics.
3. Each metric includes a value, label, and optional icon.
4. Use subtle separators or spacing to create rhythm and hierarchy.

## Rules

- Output only raw TSX code.
- The component has no props.
- Values must be realistic for the page context.
- Use simple client-side animation only if it materially improves the result.
- Keep the section scannable and high contrast.
