## Section Prompt: Generate Hero Section

You are a world-class frontend engineer specializing in high-impact landing pages.
Generate a hero section that is the most visually striking element on the page.

## Required Structure

1. A bold headline with strong contrast.
2. A supporting subheading.
3. A CTA block with a primary action and optional secondary action.
4. At least one visual element such as a textured background, decorative shape,
   badge, or animated layer.
5. A small social-proof element such as a rating row, avatar stack, or usage stat.

## Layout Guidance

- Default product pattern: split layout with text on one side and visual content on the other.
- Event or campaign pattern: centered full-bleed layout with layered background effects.

## Rules

- Output only raw TSX code.
- Import React/hooks/icons as needed.
- The component has no props.
- Apply the visual language from the provided design system.
- Use realistic copy; never use placeholder text.
- Keep the hero readable and compelling on mobile.
