# App fonts (self-hosted)

Latin subset woff2 files for the Open-OX shell UI.

| Family | Source | License |
|--------|--------|---------|
| Plus Jakarta Sans | [Fontsource](https://fontsource.org/fonts/plus-jakarta-sans) | [OFL-1.1](https://openfontlicense.org/) |
| JetBrains Mono | [Fontsource](https://fontsource.org/fonts/jetbrains-mono) | [OFL-1.1](https://openfontlicense.org/) |

Kept in-repo so `pnpm build` on deploy hosts (e.g. Tencent CVM) does not fetch `fonts.googleapis.com`.
