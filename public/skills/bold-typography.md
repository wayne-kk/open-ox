# Bold Typography Style

Design philosophy: Headline-dominant composition where large type drives structure.

## Visual Direction
- Define a signature palette that immediately signals the style in first view
- Build hierarchy through clear type scale jumps and predictable spacing rhythm
- Keep shape language consistent across cards, controls, and separators
- Use contrast intentionally to direct attention to value and action
- Apply effects sparingly so readability remains the primary objective

## Component Style
- Buttons: obvious primary/secondary distinction with strong interaction feedback
- Cards: repeatable module anatomy (title, body, meta, action) for coherence
- Navigation: concise labels, visible active state, stable alignment rules
- Forms/data blocks: clear labels, helper text, and predictable error states

## Layout
- Use a repeatable section cadence (hero -> proof -> feature detail -> CTA)
- Keep gutters and vertical rhythm consistent across breakpoints
- Prefer modular grids that scale without breaking hierarchy

## Motion & Interaction
- Use short transitions (150-250ms) for hover/focus/expand states
- Let motion explain state changes; avoid decorative-only animation

## Tone
Confident and coherent. Execute bold typography as a full system, not a one-off visual gimmick.
