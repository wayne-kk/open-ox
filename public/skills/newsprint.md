# Newsprint Style

Design philosophy: Print-inspired editorial layout with serif hierarchy, visible rhythm, and newspaper-like information density.

## Visual Direction
- Off-white paper tones with ink-like dark text
- Multi-column grid and clear typographic cadence
- Section dividers mimic publication rule lines
- Dense but organized blocks with careful hierarchy
- Use occasional stamp-like accents for focal emphasis

## Component Style
- Buttons: subtle, label-forward, publication-style controls
- Cards: article modules with metadata and bylines
- Navigation: issue-like sections instead of app chrome
- Imagery: black-and-white or muted documentary treatment

## Layout
- Use a repeatable section cadence (hero -> proof -> feature detail -> CTA)
- Keep gutters and vertical rhythm consistent across breakpoints
- Prefer modular grids that scale without breaking hierarchy

## Motion & Interaction
- Use short transitions (150-250ms) for hover/focus/expand states
- Let motion explain state changes; avoid decorative-only animation

## Tone
Editorial, intelligent, archival, and informative.
