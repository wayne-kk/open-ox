# SaaS Style

Design philosophy: Conversion-focused product marketing style optimized for clarity, trust, and fast scanning.

## Visual Direction
- Clean neutral background with one product accent
- Strong value hierarchy: headline, proof, features, CTA
- Generous spacing around pricing and trust components
- Icon-guided feature lists with concise copy blocks
- Visual consistency prioritized over visual novelty

## Component Style
- Buttons: clear primary CTA with low-friction secondary
- Cards: modular feature/pricing/testimonial blocks
- Navigation: predictable, concise, and conversion-oriented
- Forms: simple fields, obvious validation and helper text

## Layout
- Use a repeatable section cadence (hero -> proof -> feature detail -> CTA)
- Keep gutters and vertical rhythm consistent across breakpoints
- Prefer modular grids that scale without breaking hierarchy

## Motion & Interaction
- Use short transitions (150-250ms) for hover/focus/expand states
- Let motion explain state changes; avoid decorative-only animation

## Tone
Pragmatic, credible, and outcome-driven.
