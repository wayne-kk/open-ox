# Bauhaus Style

Design philosophy: Geometric modernism driven by primary colors, circles/squares/triangles, and constructivist asymmetry.

## Visual Direction
- Primary palette: red, blue, yellow with black structural lines
- Large geometric shapes as compositional anchors
- Asymmetric but balanced blocks; function-first modernism
- Strong typographic contrast using sans display + utility text
- Hard shadows and visible outline logic for each module

## Component Style
- Buttons: blocky, high-contrast fills with clear border weight
- Cards: geometric containers, visible edge and corner logic
- Navigation: minimal labels, strong spatial grouping
- Illustration: abstract geometric motifs over soft gradients

## Layout
- Use a repeatable section cadence (hero -> proof -> feature detail -> CTA)
- Keep gutters and vertical rhythm consistent across breakpoints
- Prefer modular grids that scale without breaking hierarchy

## Motion & Interaction
- Use short transitions (150-250ms) for hover/focus/expand states
- Let motion explain state changes; avoid decorative-only animation

## Tone
Functional-artistic, bold, rational, and energetic.
