# Modern Dark Style

Design philosophy: A contemporary dark UI language balancing premium polish, high readability, and restrained visual effects.

## Visual Direction
- Deep charcoal surfaces with layered dark neutrals
- Single accent family for focus and action priority
- Soft depth through subtle elevation, not heavy glow
- Readable typographic contrast tuned for long sessions
- Large hero areas with disciplined component density

## Component Style
- Buttons: luminous accent primary, muted secondary ghost
- Cards: tinted surfaces with low-noise separators
- Navigation: sticky dark shell with clear active indicators
- Data blocks: compact, legible, and status-color consistent

## Layout
- Use a repeatable section cadence (hero -> proof -> feature detail -> CTA)
- Keep gutters and vertical rhythm consistent across breakpoints
- Prefer modular grids that scale without breaking hierarchy

## Motion & Interaction
- Use short transitions (150-250ms) for hover/focus/expand states
- Let motion explain state changes; avoid decorative-only animation

## Tone
Premium, calm, technical, and trustworthy.
